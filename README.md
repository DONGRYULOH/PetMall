프로젝트 소개
----------

Pet Mall은 애완동물에 관한 상품들을 판매하는 사이트입니다. 평소에 많이 사용하는 쇼핑몰에 대한 주요 기능들을 구현함으로써 
이전에 배웠던 Spring,Java,MySQL 등의 기술들을 좀더 확실하게 학습하고자 생각하였습니다.

<br />

개발환경
-----

#### 프론트엔드
    JavaScript, Ajax, Json, Html / CSS, BootStrap 4
  
#### 백엔드
    JAVA, Spring MVC, Jsp, MyBatis
  
#### DB
    MySQL
  
#### WAS
    Tomcat 9.0
  
#### 배포
    AWS EC2
